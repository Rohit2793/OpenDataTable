Open DataTable

Column Sorting Example

Required files
jquery-2.2.4.js
simple_php_datasource.php
style.css
OpenDataTable.js
db.sql

Code

simply add data-sortable="no" data attribute inside <th> tag to disable column sort 
    
 <html>
   <head>
     <title>Open Data Table</title>
     <script src="https://code.jquery.com/jquery-2.2.4.js"></script>
     <link rel="stylesheet" type="text/css" href="style.css">
     <script src="OpenDataTable.js"></script>
     <script type="text/javascript">
    $(document).ready(function(){
      $(".OpenDataTable").OpenDataTable({  
        url:"simple_php_datasource.php",
      });
    });
     </script>
   </head>
   <body>
     <table class="OpenDataTable">
	       <thead>
		       <tr>
			        <th data-sortable="no" >Code</th>
			        <th>Name</th>
			        <th>Continent</th>
		        </tr>
	       </thead>
       <tbody></tbody>
     </table>
   </body>
 </html>
